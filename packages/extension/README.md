# BharatCode VS Code Extension

BharatCode brings a Cursor-class AI coding experience to VS Code with offline-first local models, multilingual input, and a safe edit pipeline.

## Features

- Ask / Plan / Agent / Implement modes
- Inline suggestions and edit diffs
- BharatAgent local runtime auto-start
- Model Manager for GGUF downloads
- Voice MVP: Hindi, Assamese, Tamil

## Development

- `npm install` at repo root and `gui`
- Run the extension in VS Code using the workspace launch configuration

## Privacy

Telemetry is OFF by default. Keys are stored in VS Code SecretStorage.
