# Contributing to BharatCode

Thanks for helping BharatCode.

## Development

- Node 20+
- `npm install` in repo root and `gui`
- Use VS Code launch configurations to run the extension

## Guidelines

- Keep telemetry off by default
- Preserve edit-contract safety and rollback guarantees
- Prefer offline-first workflows

## Pull Requests

Include tests for contract validation, diff apply/rollback, router decisions, and model manager behavior.
