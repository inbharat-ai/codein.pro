# CodeIn Public Beta — Known Limitations (2026-03-01)

- Whisper/gTTS/espeak: Optional setup for advanced STT/TTS; not bundled
- Browser STT: Chrome recommended for best results; other browsers may vary
- UI language: Only English UI; prompt language can be any supported by model
- Offline inference: Only llama-server is bundled; cloud APIs require user API keys
- Model updates: New model binaries must be downloaded manually for upgrades
- Linux/macOS: AppImage/DMG may require additional permissions on first run
- No automated smoke script yet; manual steps provided
- Some advanced features (multi-user, remote sync) are not included in beta
